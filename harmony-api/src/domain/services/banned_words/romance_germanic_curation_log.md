# Romance & Germanic Languages Curation Log

Curation of Shutterstock List-of-Dirty-Naughty-Obscene-and-Otherwise-Bad-Words
for fr, de, es, it, pt. Each word from the raw lists was classified as
`abuse`, `profanity`, or `remove`. This file documents every `remove` decision.
